# 🎮 Classic Games Hub

A modern, responsive gaming platform featuring a collection of classic board games and puzzle games. Built with a focus on clean UI, smooth animations, and an engaging user experience, the platform allows players to enjoy timeless games against AI or with friends while tracking their progress and achievements.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/classic-games-hub.git
```

### Navigate to the project

```bash
cd classic-games-hub
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
